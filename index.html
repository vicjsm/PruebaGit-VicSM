Configurar VSC

Pasos para configurar VSC:

  • 1- Instalamos el idioma
  • 2- Configuramos > Buscamos: editor Word Wrap > activamos (ON).
  • 3- Integramos la terminal/consola al editor, Presionamos Ctrl+Shitf+P > buscamos "terminal integrado"
  • 4- Configuramos el auto guardado > file-autoSabe
  • 5- Instalamos extensión para abrir documento directo al navegador > "open in browser"
  • 6- Instalamos extensión para servidor de desarrollo > "Live Server"
